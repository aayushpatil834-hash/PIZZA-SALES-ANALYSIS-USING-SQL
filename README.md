# 🍕 Pizza Sales SQL Analysis

## Overview

This project contains SQL queries used to analyze a Pizza Sales dataset. The analysis covers sales performance, customer ordering patterns, revenue insights, and business metrics using SQL.

Each analysis is written as a separate SQL file for easy understanding and execution.
